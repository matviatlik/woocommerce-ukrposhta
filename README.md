# woocommerce-ukrposhta
Ukrposhta shipping plugin for WooCommerce

Also adds Ukrainian Oblasts as WooCommerce states

Based on Ukrposhta rates taken from here:
http://ukrposhta.ua/dovidka/tarifi/universalni-poslugi

For international shipping (other than Ukraine) the USD rate should reflect an Ukrposhta rate:
http://ukrposhta.ua/ua/kalkulyator-forma-rozraxunku
